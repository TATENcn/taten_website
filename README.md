## 关于我们

TATEN 是一个由各地的初中生与高中生自发组织的技术交流团队。专注于计算机、开源项目与工程实践。无论是算法、前后端开发、App、游戏制作，还是服务器运维，我们都勇于尝试、持续学习。

我们相信：**年龄不是限制，兴趣和实践才是最好的老师。**
